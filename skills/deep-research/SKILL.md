# Deep Research Skill

Comprehensive research capability that explores multiple sources for thorough analysis.

## Usage

Trigger deep research mode:

```
deep-research what are the best practices for implementing OAuth2 in Node.js
```

```
investigate the performance implications of using Redis vs Memcached
```

## Research Process

### 1. Internal Exploration
- Search codebase for related implementations
- Identify existing patterns and conventions
- Find relevant configuration files

### 2. Documentation Search
- Official documentation for frameworks/libraries
- API references
- Configuration guides

### 3. Community Resources
- GitHub issues and discussions
- Stack Overflow answers
- Blog posts and tutorials

### 4. Synthesis
- Combine findings into actionable insights
- Highlight best practices
- Note potential pitfalls

## Output Format

Research results include:
- **Summary**: Key findings in 2-3 sentences
- **Details**: Comprehensive breakdown by source
- **Recommendations**: Actionable next steps
- **Sources**: Links to referenced materials

## Examples

### API Design Research
```
deep-research REST API versioning strategies and which fits our microservices architecture
```

### Performance Investigation
```
investigate why our database queries are slow and research optimization techniques
```

### Security Analysis
```
research security best practices for handling user authentication tokens
```
